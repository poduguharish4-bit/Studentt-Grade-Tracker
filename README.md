he Student Grade Tracker is a Java-based application that helps users manage and monitor student academic performance. The program allows users to enter student details and their marks in different subjects. After receiving the marks, it automatically calculates the total score, average percentage, and final grade.

Workflow
Add Student Information
The user enters the student's name, roll number, and marks for each subject.
Store Data
The program stores the student details and marks in appropriate data structures such as arrays or ArrayLists.
Calculate Results
Total Marks = Sum of all subject marks.
Average Percentage = Total Marks ÷ Number of Subjects.
Grade is assigned based on the average percentage.
Display Performance
The system shows:
Student Name
Roll Number
Subject Marks
Total Marks
Average Percentage
Grade
Manage Multiple Students
Users can add records for multiple students and view their results whenever needed.
Grade Calculation Example
Percentage	Grade
90 - 100	A+
80 - 89	A
70 - 79	B
60 - 69	C
50 - 59	D
Below 50	F
Benefits
Reduces manual calculation errors.
Provides quick and accurate grade evaluation.
Helps teachers and students track academic performance efficiently.
Simple and user-friendly interface.

Example:
If a student scores 85, 90, and 95 in three subjects:

Total Marks = 270
Average = 90%
Grade = A+
